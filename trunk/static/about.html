Google search results are NOT case sensiteve.

This application will google basic terms, scan through the results and filter out results of different case than the search terms.
Support for phrase search (quotes) has been recently added but no other advanced search option is supported like negative terms, etc.
If you want to use advanced search options then you may use the second input box to provide filter terms without the advanced options.

The initial intention was to create an application which would run a query for a longer period and the client would poll for new results in every few seconds. As the app engine can not spwan sub process the client has to wait until the query is finished and won't get any results until that. Also, the user can set the maximum number of results that will be returned and the maximum number of Google results that will be scanned through.
